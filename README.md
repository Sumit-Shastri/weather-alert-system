# 🌤️ Weather Alert System

A Python CLI tool that fetches live weather for any city
and sends smart alerts based on temperature and conditions.

## ✨ Features
- Live weather data via OpenWeatherMap API
- Temperature, humidity and condition display
- Smart alerts for extreme heat or cold
- Error handling for invalid cities
- Secure API key with .env

## 🚀 Usage
```bash
python weather.py Pune
python weather.py Mumbai
python weather.py Delhi
```

## 📤 Output

----  Weather Report for Pune  ----
Temperature : 38°C
Humidity    : 65%
Condition   : Haze
⚠️ Warning: Temperature too high, stay hydrated!
